From Terminal do the following to test receiving a byte (character) array from pyboard:

python
>>> import serial
>>> ser = serial.Serial('/dev/tty.usbmodem1452',9600)
>>> ser.readline()
'47,23000222\n'
>>> ser.readline()
'48,23500222\n'
>>> ser.readline()
'49,24000222\n'
>>> ser.readline()
'50,24500222\n'
